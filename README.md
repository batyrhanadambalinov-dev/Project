# ⏰ Deadline Tracker with Telegram Notifications

Веб-приложение на Django для отслеживания учебных дедлайнов с автоматической системой уведомлений через Telegram-бота. Проект помогает студентам контролировать сроки сдачи лабораторных работ и заданий, присылая предупреждения за день до дедлайна.

---

## 🚀 Основной функционал

* **Управление дедлайнами:** Создание, редактирование, просмотр и отметка о выполнении задач с привязкой к предметам.
* **Умный расчет времени:** Автоматическое вычисление оставшихся дней до сдачи задачи на основе текущей даты.
* **Безопасная привязка Telegram:** Генерация уникального одноразового кода авторизации в личном кабинете сайта для синхронизации с ботом.
* **Автоматические уведомления:** Фоновый скрипт проверяет базу данных и присылает сообщения в Telegram о «горящих» задачах, которые нужно сдать завтра.

---

## 🛠 Технологический стек

* **Backend:** Python 3.x, Django Framework
* **Database:** SQLite3
* **Telegram Bot API:** pyTelegramBotAPI (`telebot`)
* **Автоматизация:** Windows Task Scheduler (Планировщик задач) / `.bat` скрипты
* **Конфигурация окружения:** `python-dotenv` (безопасное хранение токенов)
